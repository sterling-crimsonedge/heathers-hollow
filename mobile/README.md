# Mobile Companion App

> **Status:** Placeholder — future work.

The idea: while the player is away from the desktop game, villagers send them "texts" about what's happening back in the hollow. Margot found a rare butterfly. The shopkeeper is closing early today. Someone's birthday is coming up.

The companion app reads from the same AI server and world state as the game client, so the village keeps living even when you're not playing.
